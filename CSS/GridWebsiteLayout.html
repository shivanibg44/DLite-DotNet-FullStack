The grid-template-areas allows us to use a kind of "ASCII art" to create each named grid area within our grids. You simply write out the grid exactly how it should appear in the browser. You do this using string values across one or more lines. Each line represents a row, and each cell within the string represents a column. You can repeat cell names to create a grid area that takes up multiple cells. This is how we did the header, footer, and nav area in the above example. So we can easily see from the above code that the grid contains 3 rows, 3 columns, 9 grid cells, and 5 named grid areas.
