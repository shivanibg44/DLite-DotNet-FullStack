BOOTSTRAP JUMBOTRON

        A Bootstrap jumbotron specifies a big box for getting extra attention to some special content or information.
        
        It is displayed as a grey box with rounded corners. It can also enlarge the font sizes of the text inside it.
        
        You can put any valid HTML or other Bootstrap elements/ classes inside a jumbotron.
        
        The class .jumbotron within the div element is used to create a jumbotron.
