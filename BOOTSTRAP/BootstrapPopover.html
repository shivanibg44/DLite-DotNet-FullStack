Bootstrap Popover

The bootstrap popover plugin is very similar to tooltips. It appears as a pop-up box when the user clicks on an element. The difference is that the popover can contain much more content. To create a popover, add the data-toggle="popover" attribute to an element and the title attribute to specify the header text of the popover, the data-content attribute to specify the text that should be displayed inside the popover's body.
