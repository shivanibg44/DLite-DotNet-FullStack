The ID Selector

The #id Selector -The jQuery #id selector uses the id attribute of an HTML tag to find the specific element. -An id should be unique within a page, so you should use the #id selector when you want to find a single, unique element. 
            
                $(document).ready(function(){
                    $("#button1").click(function(){
                        $("#test").hide();
                    });
                    });

The Class Selector

The .class Selector -The jQuery .class selector finds elements with a specific class. -To find elements with a specific class, write a period character, followed by the name of the class 
            
                
                $(document).ready(function(){
                    $("#button2").click(function(){
                      $(".test1").hide();
                    });
                  });
